# Bubble Sort

- 시간복잡도: O(n<sup>2</sup>)

- Stable Sort

- 순회를 마칠 때 마다 하나의 요소가 자신의 위치를 찾는다.  
따라서 2 회차 순회부터는 비교연산 횟수가 하나씩 감소한다.  

<br>

## Example

`[9, 8, 6, 4, 1, 3]`  

| Index | 0 | 1 | 2 | 3 | 4 | 5 |
|-------|---|---|---|---|---|---|
| Value | 9 | 8 | 6 | 4 | 1 | 3 |

## Python Implementation Code

```python
def bubble_sort(data):
    l = len(data)
    for i in range(l-1):
        for j in range(l-1-i):
            if data[j] > data[j+1]:
                data[j], data[j+1] = data[j+1], data[j]
    return data
```
